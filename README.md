# 🏦 תקציב שנתי - הוצאות והכנסות

כלי מעקב תקציב שנתי מבוסס HTML — גרסת ממשק מתקדמת לקובץ האקסל המקורי.

## תכונות

- **12 חודשים** + תצוגה שנתית מלאה
- **12 קטגוריות הוצאות** עם פירוט מלא (דיור, מזון, תחבורה, תקשורת, בריאות, פנאי, לבוש, השכלה, תרבות, חיות מחמד, פיננסים, שונות)
- **חישוב אוטומטי** — כל סיכומים מחושבים בזמן אמת (כמו נוסחאות האקסל)
- **יעד חיסכון** — הגדר % חיסכון, ראה פרוגרס בר וסטטוס מול היעד
- **גרפים** — בר גראף ודונאט בתצוגה שנתית
- **שמירה אוטומטית** — נתונים נשמרים ב-LocalStorage
- **ייצוא / ייבוא JSON** — גיבוי וטעינת נתונים

## שימוש

פתח את `index.html` בדפדפן — **לא דרוש שרת**.

או ב-GitHub Pages: `https://<username>.github.io/budget-tracker/`

## מבנה הקבצים

```
index.html   — האפליקציה המלאה (HTML + CSS + JS)
data.json    — תבנית נתונים ריקה (12 חודשים)
```

## נוסחאות (זהות לאקסל)

| חישוב | נוסחה |
|-------|-------|
| סה"כ הכנסות | SUM של 6 פריטי הכנסה |
| סיכום קטגוריה | SUM של פריטי הקטגוריה |
| סה"כ הוצאות | SUM של 12 סיכומי קטגוריות |
| חיסכון בפועל | הכנסות − הוצאות |
| יעד חיסכון | הכנסות × % יעד |
| פער מהיעד | חיסכון בפועל − יעד |
| % חיסכון | IFERROR(חיסכון / הכנסות, 0) |
